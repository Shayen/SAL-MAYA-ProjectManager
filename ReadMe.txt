Install:
1.Copy folder 'salprojectmanager' to C:\Users\<USER>\Documents\maya\<VERSION>\scripts
2.Open Autodesk Maya and open Script Editor
3.paste python command in your script editor:
'''
from salprojectmanager import SALWin
SALWin.main()
'''
4.Hilight and 'CTRL+ENTER' to run script

ps.You can drag the script with middle mouse to Maya shelf to save and reuse this script.

ENJOY
